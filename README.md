📊 Sales Data Analytics System using MySQL

📌 Project Overview

The Sales Data Analytics System is a database project developed using  MySQL to manage and analyze sales operations. It stores information about customers, products, suppliers, employees, inventory, purchases, sales, payments, returns, reviews, discounts, and branches. The project demonstrates how a retail business can efficiently manage data and generate useful business insights using SQL.

🎯 Objectives

* Manage customer and product information.
* Track sales and purchase transactions.
* Maintain inventory and stock levels.
* Manage suppliers and branch details.
* Record payments and product returns.
* Analyze sales performance using SQL queries.
* Demonstrate advanced MySQL concepts.

🛠️ Technologies Used

* MySQL Workbench

🚀 How to Run the Project

1. Install MySQL Server and MySQL Workbench.
2. Open MySQL Workbench.
3. Create a new database.
4. Import the `sales_data_analyst.sql` file.
5. Execute the SQL script.
6. Run SQL queries to explore and analyze the data.
